Animaciones 2D y 3D Interactivas

Luces dinámicas, gradientes, partículas y deformaciones. Interacción en tiempo real con el mouse.

1) Disco de gradiente / Cubo con luz móvil
2) Cinta Bézier reactiva / Esfera con spotlight
3) Partículas con repulsión / TorusKnot cromático
4) Lissajous con trail / Galaxia de puntos
5) Isométricos ilusorios / Ico deformado
6) Eclipse con glow / Malla instanciada

Pasa el mouse sobre las tarjetas para acelerar las animaciones. Mueve el mouse dentro de cada canvas para interactuar.
